微光考核Python-02:爬虫入门

难度系数：简单 相信点进这里的小伙伴之前应该听说过用 Python 写爬虫吧，那么爬虫到底是啥呢？简单来说，就是从某个网页上提取出我们所需要的特定的信息。一个简单的爬虫需要完成两件事：一是请求获取完整的网页 HTML 信息，二是通过解析 HTML 来获得我们需要的数据，接下来我们就来试着写一个简单的爬虫吧！

二.解决思路

1.网站的爬取

首先就是把自己伪装成一个浏览器。

首先在你想要爬取的网站上按F12，调出网络请求；然后暂停请求的刷新，选取最初的时间段（这样只会有一个请求，方便一些）；然后点击请求，将标头数据粘到自己的伪装信息中。

其中，电脑的配置（user-agent）最重要，通常只需要伪装这部分就可以，否则你的程序会报错，如果是Error 418就说明你被网站发现是爬虫了。

然后就可以获得网站内容了。
