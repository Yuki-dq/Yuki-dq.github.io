微光考核Python-03:爬虫翻译

难度系数：普通 想查一个单词但是得打开许多网页？不想看见杂七杂八的广告？不妨用 Python 写一个方便简洁的翻译小工具，把别人的翻译网站为我所用【仅限学习！！！】。

题目要求

爬取有道翻译（这个最简单），做一个中英翻译小软件。

拓展部分

为你的翻译工具写一个简单的界面，并且让那个丑陋的黑框框消失。

二.解决思路

1.网站的爬取

首先打开有道翻译的网站，找到translate_o?一项点开，将表单数据中的数据放在urllib.request.Request()的data属性中。其中‘i’对应的就是想要翻译的字符。

2.网站的解析

解析之后会得到这样一条信息：

{“type”:”EN2ZH_CN”,”errorCode”:0,”elapsedTime”:1,”translateResult”:[[{“src”:”life”,”tgt”:”生活”}]]}

其中tgt对应的是翻译网站返回的翻译结果。

3.将数据转换成界面

总共四部分，从上到下：标签、输入框、标签、输出框，然后绑定回车键。

三.一些问题

1.翻译结果的提取

需要注意的是，返回的信息并不是字典而是**’bs4.BeautifulSoup’**，所以并不能用key值查询，也转换不成字典形式。

本人的方法是将这一串转换成str类型，然后用re提取tgt后面的一串字符，最后从第7位开始提取字符合并。

2.爬取的网址

爬取的网址中如果有translate_o一项，可以把_o去掉，否则会出现**”Errorcode: 50”**错误。

3.tkinter的输入栏

输入栏可以用**tk.Entry()加get()的组合读取输入的东西，也可以用tk.Test()和get(起,止)**的组合，不过第一种会让输入栏变得非常扁导致看起来不好看。不扁我的翻译框也看起来好丑啊
